## Getting Started

### Prerequisites
- Python 3.10+
- pip or uv for package management

### Development Setup

```bash
# Clone the repository
git clone https://github.com/templetwo/sovereign-stack.git
cd sovereign-stack

# Install in development mode
pip install -e .

# Run tests
python tests/test_integration.py
```

## Code Style

### General Principles
- **Simplicity over cleverness** - Code should be readable
- **Minimal dependencies** - Only add what's truly necessary
- **Document the why** - Comments explain reasoning, not mechanics

### Python Style
- Follow PEP 8
- Use type hints where they add clarity
- Keep functions focused and small

### Commit Messages
- Use present tense ("Add feature" not "Added feature")
- Keep the first line under 72 characters
- Reference issues when applicable
